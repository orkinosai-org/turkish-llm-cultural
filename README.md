# turkish-llm-cultural
# Turkish Cultural LLM (HIT-30 Project)

This project aims to build a sovereign Turkish language model trained on historical, literary, and Islamic texts such as Mesnevi, Yunus Emre poetry, and Risale-i Nur. It is designed to support cultural preservation, education, and spiritual dialogue in collaboration with universities and Diyanet.

## Goals
- Train a Mistral-based Turkish LLM using open-source models
- Fine-tune on culturally significant Turkish texts
- Deploy on Azure GPU infrastructure
- Align with HIT-30 national AI strategy
